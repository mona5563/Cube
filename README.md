- C言語

## ゲーム概要
3×3×3のボックスをスライドさせ、数字の描かれたボックスを
旗の描かれた場所まで運ぶ立体スライドパズル

## 操作説明
- ゲームスタート：マウス左クリック
- ポーズ・ゲーム説明：TABキー
- ボックスの移動：マウス左クリック
- ボックスの回転：マウス右ドラッグ
